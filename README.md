# Dispensary Management System

## Project Overview

The Dispensary Management System (DMS) is a database-oriented system designed to manage and organize the routine operations of a medical dispensary. The system maintains structured records of patients, doctors, medicines, prescriptions, and billing information to ensure efficient data handling and reduced manual effort.

This assignment includes:
- Database Design
- ER (Entity-Relationship) Diagram
- UML Diagrams

---

## Objectives

- To design a structured relational database for a dispensary system.
- To represent system entities and their relationships using an ER diagram.
- To model the functional aspects of the system using UML diagrams.
- To understand database normalization and relational schema design.

---

## Features

- Patient record management
- Doctor information management
- Appointment scheduling
- Prescription record maintenance
- Medicine inventory tracking
- Billing management
